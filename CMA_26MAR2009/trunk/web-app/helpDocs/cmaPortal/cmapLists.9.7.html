Navigating the Sidebar Lists
Each of the lists in the sidebar contain predefined and custom (if you have defined them) lists. For example, the following figure illustrates the PatientDID List section of the sidebar. Default CMA Portal lists are displayed in black text; custom lists you may have created in your current session are displayed in red text. See Creating Custom Lists.
To display the data included in analysis, hover your cursor over the list name of interest.
A popup lists the data items.
For example, the following figure displays a partial list of patientDIDs for the EFGR Somatic Mutation (validated) list.
To export a list to a spreadsheet, double-click the list name. The CMA Portal system launches a spreadsheet application and displays the data in a worksheet.
