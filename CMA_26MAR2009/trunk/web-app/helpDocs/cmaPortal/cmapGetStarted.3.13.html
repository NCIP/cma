Navigating the Cancer Molecular Analysis Portal Workspaces
Each of the workspaces, Gene View, Genome View, Clinical View, and the Analysis Tools has its own workspace which is displayed whenever you select an item on the Home page menu bar. Each workspace contains tabbed pages, a side bar, and unique tools and resources. The tabs and sidebar are the same for all workspaces except the Login/Register workspace.
