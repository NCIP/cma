Navigation Pane Links
Each section of the Navigation pane acts as a link to its respective workspace. Click a link to access the tools and resources associated with each workspace.
The following table describes each item on the Navigation pane.
Callout
Number
Description/Function
1
Gene View link – For TCGA and REMBRANDT data only, provides access to the gene analysis workspace where you can visualize gene expression, copy number, SNP, and pathway data on a gene by gene basis; and generate detailed study-related reports for a given gene.
2
Genome View link – For TCGA and REMBRANDT data only, provides access to the genome workspace where you can explore all of the study data in one genome level visualization, and investigate chromosomal regions of amplification, deletion, and over expression.
3
Clinical View link – Provides access to the clinical analysis workspace where you can investigate and study clinical data, and explore the relationships between clinical and molecular study data.
4
Analysis Tools link – For TCGA and REMBRANDT data only, provides access to the analysis tools workspace, where you can analyze the study data using analysis tools such as GenePattern, Principal Component Analysis, and the CGA Workbench.
