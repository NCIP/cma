Accessing the Data Access Matrix
The Data Access Matrix (the Matrix) application enables researchers to select data sets in TCGA (The Cancer Genome Atlas) servers through a user-friendly graphic-based selection system.
How to Access the Data Access Matrix
1.
On the sidebar, click the Cancer Genome Atlas Data Download link.
The Cancer Genome Atlas Data Portal appears in a new browser window.
2.
Click the Browse Data tab on the top of any Portal page.
3.
Under Search by Sample, click the Cancer type selector (down arrow) and then select the cancer type of interest.
Data for ovarian cancer tissue (OV) is not available in this release.
4.
Click the Search by Samplelink.
You can also click anywhere in the Data Access Matrix image.
Follow the instructions in the Data Access Matrix documentation for selecting and downloading data.
