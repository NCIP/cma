Viewing and Managing List Content
CMA Portal enables you to manage the content of predefined and custom lists as follows:
•
View details of a list’s content. See Viewing List Details
•
View a list’s creation date and notes. See Viewing List Creation Data and Notes
•
Delete a list. See Deleting Lists
•
Delete an item from a list. See Deleting Items From a List
•
Export a list. See Exporting Lists
