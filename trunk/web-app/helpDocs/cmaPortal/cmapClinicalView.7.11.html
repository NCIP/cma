Creating a PatientDID List From a Sample-based Clinical Report
From the Clinical Report page, you can select patients and save them to a customized PatientDID list to use as criteria in future queries.
How to Create a PatientDID List
1.
Select one or more patients to save to the list. See Step 1
2.
To save the selected records to a PatientDID list, type a unique name in the mySavedList box, and click Save Patient List.
The new sample set is listed in red text in the PatientDID List in the side bar.
