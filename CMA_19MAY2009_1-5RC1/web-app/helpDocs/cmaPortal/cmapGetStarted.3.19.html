PatientDID List Section Features
De-identified ID (DID) information is data from tissue sample analysis from which personally identifiable information has been removed such that a researcher can no longer trace data back to an individual patient. For a description of PatientDIDs and the PatientDID lists, see About PatientDID Lists and Navigating the Sidebar Lists.
