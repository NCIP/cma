• Data Version [+/-]
    The TARGET data on this site includes data that was authorized for release as of March 2009. This includes clinical data for 255 acute lymphoblastic leukemia patients, as well as genomic copy number data and validated somatic mutation data for a small number of genes. Gene expression data will be released in the near future. A description of the TARGET project can be found at http://target.cancer.gov.
