Specifying Clinical Data Search Criteria
CMAP provides several tools for querying, visualizing, and downloading patient and sample data. You can search for data directly from the Clinical View workspace or via the Data Access Matrix. For instructions on accessing the Data Access Matrix, see Accessing TCGA Data.
The Clinical View workspace provides different search criteria depending on the context you are working with, TCGA or REMBRANDT.
