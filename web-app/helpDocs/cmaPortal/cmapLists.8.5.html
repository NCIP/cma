About Gene Lists
Each list in the Genes Lists section on the Manage Lists page contains a set of genes of interest. Currently you can use either default TCGA or REMBRANDT genes, or custom lists to filter your queries.
Currently the available data displayed is provided by TCGA Centers as of a given date. When working in the REMBRANDT context, the data displayed is provided by REMBRANDT collaborators. Other gene targets will be available in subsequent releases of this portal.
You can download TCGA gene lists at:
http://gforge.nci.nih.gov/docman/view.php/259/7051/TCGA%20target%20lists.xls
