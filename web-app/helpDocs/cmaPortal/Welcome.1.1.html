Cancer Molecular Analysis Portal Online Help
Welcome to the Cancer Molecular Analysis Portal Online Help. To start using the Cancer Molecular Analysis Portal, select from the following topics:
•
Getting Started With the Cancer Molecular Analysis Portal provides information for logging in to and navigating the Cancer Molecular Analysis Portal.
•
Working With Gene-Based Views describes how to use the Cancer Molecular Analysis Portal to research gene expression, copy number, SNP, and pathway data.
•
Working With Pathways and Associated Anomalies describes how to use the Cancer Molecular Analysis Portal to review pathways and the anomalies associated with them.
•
Working With Genome Views describes how to use the Cancer Molecular Analysis Portal to investigate chromosomal regions of amplification, deletion, and over-expression.
•
Working With Clinical Views describes how to use the Cancer Molecular Analysis Portal to study clinical data and explore the relationships between clinical and molecular study data.
•
Analysis Tools describes how to use the Cancer Molecular Analysis Portal to perform principal component and gene pattern analyses, and to access associated analysis applications.
•
Managing Lists describes how to manage patient and/or gene lists by editing application-defined lists and creating new custom lists.
•
Glossary provides definitions of acronyms, abbreviations, and terminology.
•
Application Support provides contact information for general application support.
To learn more about each topic, browse through the online help or use the table of contents, index, or search features of the help. For more information, see Using the Online Help.
You can also find comprehensive procedures for all Cancer Molecular Analysis Portal tasks in the Cancer Molecular Analysis Portal User’s Guide.
