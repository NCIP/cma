Administration Section Features
The following table describes each feature in the Administration section.
Feature
Description/Function
View Results
Provides access to the workspace in which you have saved queries
List Management
Provides access to the default and any custom lists you may have created
Cancer Genome Atlas Data Download
Links to The Cancer Genome Atlas Data (TCGA) portal from which you can search for and download data from TCGA datastores. Also provides access to the Data Access Matrix.
