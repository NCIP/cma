Filtering Clinical Search Criteria
The Clinical Query Filter page enables you to refine your query.
How to Specify Search Criteria
1.
Follow the instructions in Specifying Clinical Search Criteria to select the groups of data you want to query in order to access the list of filters.
2.
On the Criteria Filter page, in the Select Patient List, select a patient group to query. Available groups of patients with certain characteristics are the same as those listed in the PatientDID Lists shown on the sidebar. If you created and added a custom list, the list name appears in red text in the list of patients.
3.
Continue to select criteria from the options displayed. The number/type of criteria vary according to the groups you selected in your initial query criteria selection. See Specifying Clinical Search Criteria
4.
In the Name Analysis Result box, type a unique name for the query.
You will lose prior query data if you enter the same name as a previous query during the current session.
5.
To clear the values entered on the page and enter new values, click Clear.
6.
To submit the query, click Submit.
CMA Portal processes your query and displays completed when the result has been returned.
The CMA Portal stores the query for the duration of your current CMA Portal session.
7.
To view the results of a query, click the query name (link).
The results appear in a new browser window.
For information about the results, see Viewing Clinical Reports.
