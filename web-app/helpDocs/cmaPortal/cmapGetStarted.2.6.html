Navigating the Cancer Molecular Analysis Portal Home Page
The Cancer Molecular Analysis Portal home page provides links and other information that enable you to navigate through CMA Portal workspaces.
The following table describes each of the panes on the Home page, from left to right, and the Context drop-down list at the upper right side of the page.
Area
Description/Function
Navigation pane
Provides links to query and analysis workspaces
Overview pane
Provides a brief overview of each of the workspaces
User Access pane
Login page for existing users. Provides links to register for an account and to provide feedback and request CMAP support.
Context
Indicates the data set used for your queries. Currently data from TCGA and REMBRANDT is available for research. To change the context, click the arrow next to the Context drop-down list, and select the data set of interest.
