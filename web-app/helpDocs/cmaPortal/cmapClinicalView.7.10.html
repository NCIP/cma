Selecting and Downloading Sample-based Clinical Report Records
How to Select and Download Report Records
1.
Do one of the following to select sample records for download:
•
To select individual records, click the check box beside each desired record.
-or-
•
To select all records, select the Check All check box. This check box acts like a toggle switch. To clear all selected records, select the Check All check box again.
2.
To save the list of patient records that you selected, type a unique name in the mySavedList box, and click Save Patient List.
3.
To download a comma-separated value file containing the selected records, click Download CSV. When prompted, open or save the file to your computer.
4.
To close the report, click Close.
