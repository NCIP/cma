Navigating the Cancer Molecular Analysis Portal Workspaces
The Gene View, Genome View, Clinical View, and the Analysis Tools workspaces are displayed whenever you select an item on the Home page. Each workspace contains tabbed pages, a side bar, and unique tools and resources. The tabs and sidebar are the same for all workspaces.
