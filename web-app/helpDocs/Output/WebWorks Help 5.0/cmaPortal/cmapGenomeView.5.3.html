Viewing Copy Number Data
Differences in the number of copies of certain genes contributes to genetic variability. Variability can be caused by deletions of some genes on only one chromosome, or multiple copies of some genes.
Copy number data in the CMA Portal is provided by the following sources:
•
Broad Institute (Broad)
•
Harvard Medical School (HMS)
•
Memorial Sloan-Kettering Cancer Center (MSKCC)
•
HudsonAlpha Institute for Biotechnology (HAIB)
Note:
Copy numbers are also known as copy number variants (CNVs) and copy number polymorphisms (CNPs)
For more info on characterization sources, refer to TCGA site at:
http://cancergenome.nih.gov/data/types/genomic/
